# AWS Development & Deployment Guide

This guide covers AWS services integration with Django applications, from configuration through deployment at scale.

## Course Information
- [Python Django for AWS Development - Mastery course - Part 1](https://www.udemy.com/course/python-django-for-aws-development-mastery-course-part-1)

## Learning Path

This guide progresses through AWS integration phases:

1. **[AWS Configuration](section-5-6.md)** - Setting up AWS services and IAM roles
2. **[AWS Integration Phase-I](section-7.md)** - Integrating AWS services with Django (S3, RDS, Route53)
3. **[AWS Integration Phase-II](section-8.md)** - Advanced deployments with Elastic Beanstalk and SES

Each section includes configuration examples, deployment strategies, and production considerations.
